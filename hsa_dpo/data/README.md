Datasets of [HSA-DPO](https://github.com/Mr-Loevan/HSA-DPO)

For hallucination detection, image is from [Visual Genome](https://homes.cs.washington.edu/~ranjay/visualgenome/api.html), training dataset is in `hsa_dpo_detection.jsonl`

For hallucination mitigation, image is in `hsa_dpo_imgs.tar.gz`, preferences dataset is in `hsa_dpo_preference_llava1dot5.jsonl`. Note that llava1dot5 means 'rejected' is generated by llava-v1.5.
